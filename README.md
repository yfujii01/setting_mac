# setting_mac

```sh
# brew
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

# cask
brew tap homebrew/cask

# chrome
brew cask install google-chrome

# ssh setting
mkdir ~/.ssh

# 中身はLastPassから
vi ~/.ssh/id_rsa

chmod 600 ~/.ssh/id_rsa 

# ime
brew cask install google-japanese-ime
## OS再起動
sudo reboot

# PC名変更
# システム環境設定 > 共通 > コンピュータ名


```
